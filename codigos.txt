Iniciar VUE: 'yarn serve' ou 'npm run serve'
Iniciar Laravel: 'php artisan serve'

Explicação node_modules: https://prnt.sc/26jqoqk

Caso baixe o projeto pelo GitHub, utilize o comando: 'npm install' ou 'yarn install'
para que possa ser baixado todas as dependencias necessárias para o projeto rodar.

Para saber todos os comandos do laravel: 'php artisan'
Para criar controller: 'php artisan make:controller <nome da controller>'
Para criar model: 'php artisan make:model <nome da controller> -m'
